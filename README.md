## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/roxyai.git
   ```

2. Navigate to the project directory:

   ```bash
   cd roxyai/roxy
   ```

3. Install dependencies:

   ```bash
   flutter pub get
   ```

4. Run the application:

   ```bash
   flutter run
   ```
